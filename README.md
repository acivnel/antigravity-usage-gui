# Antigravity Usage GUI

Приложение для мониторинга квот ваших AI моделей в Antigravity. В отличие от CLI-версии, оно работает в системном трее Windows и показывает наглядный графический интерфейс.

Этот проект полностью основан на оригинальном инструменте **[antigravity-usage](https://github.com/skainguyen1412/antigravity-usage)** от разработчика **[skainguyen1412](https://github.com/skainguyen1412)**. Огромная благодарность автору за основу и логику работы с API!

![Скриншот приложения](img.png)

## Особенности

- 🌐 **Авторизация через Google**: безопасный вход с поддержкой нескольких аккаунтов.
- 💻 **Локальное подключение**: автообнаружение запущенных IDE (VS Code, JetBrains) для мгновенного получения квот.
- 📊 **Дашборд**: отслеживание остатка квот для Gemini, Claude и GPT в реальном времени.
- 📥 **Автообновления**: приложение умеет обновляться через GitHub Releases.
- 🇷🇺 **Русский язык**: интерфейс и тексты на русском.
- 🎈 **Системный трей**: приложение всегда под рукой и не занимает место на панели задач.

## Установка и запуск

### Скачать готовое приложение (рекомендуется)

Скачайте последнюю версию со страницы [Releases](https://github.com/acivnel/antigravity-usage-gui/releases/latest).

Обычно в релизе есть:

- установщик `Antigravity Usage Setup X.Y.Z.exe`
- портативная версия `Antigravity Usage-X.Y.Z-win.zip`

### Запуск из исходников (для разработчиков)

Если вы хотите собрать приложение самостоятельно:

1.  Клонируйте репозиторий:
    ```bash
    git clone https://github.com/acivnel/antigravity-usage-gui.git
    ```
2.  Установите зависимости:
    ```bash
    npm ci
    ```
3.  Запустите в режиме разработки:
    ```bash
    npm run dev
    ```

### Сборка (.exe и .zip)

Чтобы собрать дистрибутив для Windows (NSIS installer + portable zip):

1.  Установите зависимости:
    ```bash
    npm ci
    ```
2.  Выполните команду:
    ```bash
    npm run build:win
    ```
3.  Готовые файлы появятся в папке `dist/`.
